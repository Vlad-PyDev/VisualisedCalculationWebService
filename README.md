# VisualisedCalculationWebService

This web service is simple visualised calculator on golang

# To Dowload

Dowload zip-file and unpack him

# To Setup

1. Install Python
2. Install lib - github.com/Knetic/govaluate

```bash
go get github.com/Knetic/govaluate
```

# To Run

Write in cmd

```bash
go run main.go
```
